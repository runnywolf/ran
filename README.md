# 功能

歷屆試題頁面：
- 查詢特定學校和年份的數學考卷
- 每一題都盡量提供詳細的解答
- 測驗模式下會隱藏解答，並且提供計時功能
- 提供題本的來源連結

模擬室頁面：( 解答生成器 )
- 齊次 / 非齊次遞迴
