[![](https://img.shields.io/badge/Vue.js-345?style=flat&logo=vuedotjs&logoColor=4FC08D)](https://github.com/vuejs/core)

一款用於準備資工所數學的網頁 App。<br>
[Ran](https://runnywolf.github.io/ran/#/exam)

# 功能

歷屆試題頁面：
- 查詢特定學校和年份的數學考卷
- 每一題都盡量提供詳細的解答
- 測驗模式下會隱藏解答，並且提供計時功能
- 提供題本的來源連結
